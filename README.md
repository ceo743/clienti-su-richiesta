# ⚠️ clienti-su-richiesta — cosa va qui e cosa NO

Questo repo pubblica `clienti.davidecaiazzo.it`.

## Cosa ci sta
- Il video funnel (landing, application, thank-you)
- Le sue pagine di supporto strette (`prenota-call.html`, `diagnosi-linkedin.html`)

## Cosa NON ci deve stare
- **Le guide** (newsletter del martedì o lead-magnet DM): ognuna ha il **suo repo dedicato** + sottodominio (`guida-x.davidecaiazzo.it`). MAI qui dentro, nemmeno per "fare veloce".
- **La dashboard outreach clienti** (`dashboard-outreach/`): è già finita qui per errore, resta fino alla migrazione (poi si elimina), non aggiungerci nient'altro.
